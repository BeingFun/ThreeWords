# ThreeWords

## 初衷：成年人得学会给自己灌鸡汤

## 1. ThreeWords 简介

ThreeWords 是一款为 Windows 系统桌面背景添加一句文字的桌面壁纸软件，可以自定义你感兴趣的文字分类,如动漫、文学、哲学、游戏、诗词、网易云音乐热评等。  
此外也支持设置必应每日壁纸为桌面背景功能。
